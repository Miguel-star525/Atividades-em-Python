# Sistema de Gestão com Listas

Conjunto de atividades relacionados a vida de um aluno.


## Lista de Atividades

1. **Sistema de Cadastro de Alunos**
2. **Controle de Despesas Pessoais do aluno**
3. **Cardápio de Lanchonete que o aluno frequenta**
4. **Controle de Estoque da Lanchonete**

## Tecnologias utilizadas
* Python
* Google Colab
